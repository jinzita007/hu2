从蓝猫动画代工到做原创动漫 IP

1970-01-01 08:00:00

近日，二次元 IP 制作商两点十分宣布获得数千万人民币 A 轮融资，由峰瑞资本独家投资。在做原创动漫 IP 前，他们曾参与过 “熊出没 ”、“蓝猫” 等动画作品的制作代工。


“两点十分 ” 从 14年 开始原创动漫内容的制作，目前其重点动漫 IP 主打的是都是二次元游戏人群：漫画《银之守墓人》是一个主角穿越到游戏中的故事，主推的动画爆弹精英的背景设定则跟暴雪出品的风暴英雄一致。这两部作品中，仍在腾讯动漫连载的银之守墓人基本常年排在榜单前十，《爆弹精英》上线不久就跟暴雪官方达成了合作。

关于为什么会选游戏题材作品切入，主要的原因有两点：首先其主打的二次元游戏受众已经有良好的消费习惯了，另外 IP 的游戏化衍生是他们非常看重的一个点。

此外，其作品也在对用户做正版消费潜移默化的引导，拿 “守墓人” 举例，由于大的设定框架是在一个游戏中，故事中的主角为喜欢的东西付费是常态。“喜欢这个 IP 的用户受作品的影响去为各种衍生买单会是一件很自然的事情， ” 创始人王世勇告诉我。

目前 “两点十分 ” 主推的 IP 是正在连载的银之守墓人，其动画制作的计划已经确定，跟之前大热的从前有座灵剑山类似，守墓人的动画制作也会有日本的动画制作团队参与进来，游戏化的衍生也正在推进，“希望能做成一款 VR 游戏， ” 王世勇表示。

拿到本轮融资后，除了继续运营已有 IP，“两点十分 ” 已经确定了三个新的 IP 计划：2 部漫画和一部动画。两部漫画 IP 中会有一部会跟银之守墓是相同的世界设定，但不同的故事，仍旧主打游戏化衍生，另一漫画作品的衍生重点方向将是电影，一部中国传统元素 + 美漫风格的作品。网站建设。而今年要做的新动画走的是萌系女生向的风格。

团队方面，“两点十分 ” 分为 4 个 IP 制作部门和一个市场部门，共 80 人。其漫画制作的风格偏日漫，而动画部门的制作风格则是美漫风格，这也是他们在做了漫画后没有亲自操刀去做自己 IP 的动画化的原因所在，这次融资后也会跟一些机构合作开始他们的人才培养计划，将各种风格人才的空缺补上。
